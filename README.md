# Los In-Nombrables
## Proyecto: Plataforma de Galería de Fotos
### Se requiere una plataforma que permita crear álbum de fotos, cada foto puede tener una descripción, ser comentada y tener una  valoración. El álbum  puede  tener  una descripción general. También debe tener la opción de zoom(+)/(-) para cada foto. Para crear un álbum es requisito estar registrado. Por lo que, las fotos son propiedad del usuario registrado, eso indica que debe aparecer el autor del álbum. También cada foto puede tener uno o más etiquetas, eso quiere decir que debe permitir listar las fotos que se encuentran agrupadas por etiquetas. Como usuario registrado también puedo seguir las publicaciones deotros usuarios, es decir  que  debo  recibir alerta  o  notificaciones  que  me  informen  si  hay  una  nueva publicación. 
### Debe existir las siguientes categorías: fotos más visitas, fotos mejor calificadas. También un buscador, cuya funcionalidad es buscar por álbum, foto o usuario.
